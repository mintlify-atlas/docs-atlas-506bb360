## What is OpenBB?

OpenBB operates as the "connect once, consume everywhere" infrastructure layer for financial data. A single integration exposes your data to:

- **Python environments** — for quants and data scientists via the `obb` object
- **REST API** — for apps and dashboards via `openbb-api` (FastAPI + Uvicorn)
- **CLI** — for interactive exploration via `openbb-cli`
- **MCP server** — for AI agents via `openbb-mcp`
- **OpenBB Workspace** — for enterprise dashboards and AI copilots

## Get started in 3 steps

<Steps>
  <Step title="Install the package">
    ```bash
    pip install openbb
    ```
  </Step>
  <Step title="Import and query data">
    ```python
    from openbb import obb

    # Get historical equity prices
    result = obb.equity.price.historical("AAPL")
    df = result.to_dataframe()
    print(df.head())
    ```
  </Step>
  <Step title="Explore more data">
    ```python
    # Economic indicators
    gdp = obb.economy.indicators(symbol="GDP", provider="fred")

    # Options chains
    chains = obb.derivatives.options.chains("AAPL", provider="cboe")

    # Crypto prices
    btc = obb.crypto.price.historical("BTCUSD", provider="yfinance")
    ```
  </Step>
</Steps>
