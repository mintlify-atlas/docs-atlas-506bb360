## Price history

### `obb.index.price.historical`

Historical price levels for an index by symbol. Symbol format varies by provider.

<CodeGroup>

```python S&P 500 — FMP
res = obb.index.price.historical(symbol="^GSPC", provider="fmp")
```

```python S&P 500 — Intrinio
res = obb.index.price.historical(symbol="SPX", provider="intrinio")
```

</CodeGroup>

<Note>
Not all providers use the same symbol format. Use `obb.index.available` or `obb.index.search` to confirm the correct symbol for your provider.
</Note>

## Constituents and sectors

### `obb.index.constituents`

Get the current constituent members of a given index.

<CodeGroup>

```python Dow Jones — FMP
res = obb.index.constituents(symbol="dowjones", provider="fmp")
```

```python CBOE index by ticker symbol
res = obb.index.constituents(symbol="BEP50P", provider="cboe")
```

</CodeGroup>

### `obb.index.sectors`

Sector weighting of an index. Shows how the index's market cap is distributed across GICS sectors.

```python
res = obb.index.sectors(symbol="^TX60", provider="tmx")
```

## Discovery

### `obb.index.available`

List all indices available from a given provider, with their names and symbols.

<CodeGroup>

```python FMP
res = obb.index.available(provider="fmp")
```

```python Yahoo Finance
res = obb.index.available(provider="yfinance")
```

</CodeGroup>

### `obb.index.search`

Filter the provider's index universe for rows matching a query string.

<CodeGroup>

```python All CBOE indices
res = obb.index.search(provider="cboe")
```

```python Search by symbol keyword
res = obb.index.search(query="SPX", provider="cboe")
```

</CodeGroup>

### `obb.index.snapshots`

Current levels for all indices from a provider, optionally filtered by region.

<CodeGroup>

```python TMX — Canadian indices
res = obb.index.snapshots(provider="tmx")
```

```python CBOE — US indices
res = obb.index.snapshots(region="us", provider="cboe")
```

</CodeGroup>

## S&P 500 multiples

### `obb.index.sp500_multiples`

Historical S&P 500 valuation multiples including Shiller PE (CAPE), earnings yield, dividend yield, and price-to-book ratios.

<CodeGroup>

```python All multiples
res = obb.index.sp500_multiples(provider="multpl")
```

```python Shiller PE — annual
res = obb.index.sp500_multiples(
    series_name="shiller_pe_year",
    provider="multpl",
)
```

</CodeGroup>
