Commands for accessing analyst estimates, price targets, and forward-looking financial projections. All commands are available under the `obb.equity.estimates` namespace.

<Note>
  Forward estimates commands accept optional `fiscal_year` and `fiscal_period` parameters to filter results to a specific period. When omitted, all available periods are returned.
</Note>

---

## Price targets

### `obb.equity.estimates.price_target`

Get analyst price targets for a given company.

**REST path:** `GET /api/v1/equity/estimates/price_target`

**Providers:** `benzinga`, `fmp`

<ParamField query="symbol" type="string">
  Stock ticker symbol (e.g. `AAPL`). Optional â€” when omitted, returns targets across all covered tickers.
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of records to return.
</ParamField>

<ParamField query="start_date" type="string">
  Start date in `YYYY-MM-DD` format to filter published price targets.
</ParamField>

<ParamField query="end_date" type="string">
  End date in `YYYY-MM-DD` format to filter published price targets.
</ParamField>

<ParamField query="action" type="string">
  Filter by analyst rating action (provider-specific). Example: `downgrades`, `upgrades`, `initiates`.
</ParamField>

<ParamField query="provider" type="string">
  Data provider to use.
</ParamField>

<CodeGroup>

```python Python
from openbb import obb

# All recent price targets
result = obb.equity.estimates.price_target(provider="benzinga")

# MSFT downgrades between two dates
result = obb.equity.estimates.price_target(
    symbol="msft",
    start_date="2020-01-01",
    end_date="2024-02-16",
    limit=10,
    action="downgrades",
    provider="benzinga",
)
df = result.to_df()
```

```bash REST API
curl "http://127.0.0.1:6900/api/v1/equity/estimates/price_target?symbol=AAPL&provider=fmp"
```

</CodeGroup>

**Standard response fields:** `published_date`, `published_time`, `symbol`, `exchange`, `company_name`, `analyst_name`, `analyst_firm`, `currency`, `price_target`, `adj_price_target`, `price_target_previous`, `price_when_posted`, `rating_current`, `rating_previous`, `action`

### `obb.equity.estimates.consensus`

Get the consensus price target and analyst recommendation for a given company.

**REST path:** `GET /api/v1/equity/estimates/consensus`

**Providers:** `fmp`, `yfinance`

<ParamField query="symbol" type="string" required>
  Stock ticker symbol. Accepts comma-separated values for multiple symbols.
</ParamField>

<ParamField query="provider" type="string">
  Data provider to use.
</ParamField>

<CodeGroup>

```python Python

result = obb.equity.estimates.consensus(symbol="AAPL", provider="fmp")

# Multiple symbols
result = obb.equity.estimates.consensus(symbol="AAPL,MSFT", provider="yfinance")
df = result.to_df()
```

```bash REST API
curl "http://127.0.0.1:6900/api/v1/equity/estimates/consensus?symbol=AAPL&provider=fmp"
```

</CodeGroup>

## Analyst search

**REST path:** `GET /api/v1/equity/estimates/analyst_search`

**Providers:** `benzinga`

<ParamField query="firm_name" type="string">
  Name of the analyst firm to search for (e.g. `Wedbush`).
</ParamField>

</ParamField>

<CodeGroup>
