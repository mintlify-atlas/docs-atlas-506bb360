Commands for accessing a company's financial statements and fundamental data. All commands are available under the `obb.equity.fundamental` namespace.

<Note>
  Most commands accept a `period` parameter (`annual` or `quarter`) and a `limit` parameter controlling the number of periods returned. Provider availability varies per command.
</Note>

## Financial statements

### `obb.equity.fundamental.balance`

Get the balance sheet for a given company.

**REST path:** `GET /api/v1/equity/fundamental/balance`

**Providers:** `fmp`, `intrinio`, `yfinance`

<ParamField query="symbol" type="string" required>
  Stock ticker symbol (e.g. `AAPL`).
</ParamField>

<ParamField query="period" type="string" default="annual">
  Reporting period. Options: `annual`, `quarter`.
</ParamField>

<ParamField query="limit" type="number">
  Number of periods to return.
</ParamField>

<ParamField query="provider" type="string">
  Data provider to use.
</ParamField>

<CodeGroup>

```python Python
from openbb import obb

# Annual balance sheet (default)
result = obb.equity.fundamental.balance(symbol="AAPL", provider="fmp")

# Last 5 annual periods via Intrinio
result = obb.equity.fundamental.balance(
    symbol="AAPL",
    period="annual",
    limit=5,
    provider="intrinio",
)

df = result.to_df()
```

```bash REST API
curl "http://127.0.0.1:6900/api/v1/equity/fundamental/balance?symbol=AAPL&provider=fmp"
```

</CodeGroup>

**Standard response fields:** `period_ending`, `fiscal_period`, `fiscal_year`

### `obb.equity.fundamental.management_discussion_analysis`

Get the Management Discussion & Analysis (MD&A) section from a company's financial filings.

**REST path:** `GET /api/v1/equity/fundamental/management_discussion_analysis`

**Providers:** `sec`

<ParamField query="symbol" type="string" required>
  Stock ticker symbol.
</ParamField>

<ParamField query="calendar_year" type="number">
  Calendar year of the filing period (e.g. `2020`).
</ParamField>

<ParamField query="calendar_period" type="string">
  Calendar period of the filing. Options: `Q1`, `Q2`, `Q3`, `Q4`, `annual`.
</ParamField>

<ParamField query="include_tables" type="boolean" default="false">
  Attempt to extract all tables in the MD&A as valid Markdown.
</ParamField>

<ParamField query="raw_html" type="boolean" default="false">
  Return the raw HTML file instead of extracting content. Useful for custom parsing.
</ParamField>

<ParamField query="provider" type="string">
  Data provider to use.
</ParamField>

<CodeGroup>

```python Python
from openbb import obb

# Most recent MD&A
result = obb.equity.fundamental.management_discussion_analysis(
    symbol="AAPL",
    provider="sec",
)

# Specific period with tables
result = obb.equity.fundamental.management_discussion_analysis(
    symbol="AAPL",
    calendar_year=2020,
    calendar_period="Q4",
    include_tables=True,
    provider="sec",
)

print(result.results[0].content)
```

```bash REST API
curl "http://127.0.0.1:6900/api/v1/equity/fundamental/management_discussion_analysis?symbol=AAPL&provider=sec"
```

</CodeGroup>
