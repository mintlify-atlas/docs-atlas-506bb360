The `obb.economy` namespace provides access to a broad range of macroeconomic data. Commands cover consumer prices, labor markets, GDP, trade flows, central bank activity, surveys, and global shipping — sourced from authoritative providers including FRED, OECD, IMF, EconDB, and the Federal Reserve.

## Indicators

### `obb.economy.indicators`

Get economic indicators by country and indicator symbol. Supports EconDB's indicator codes as well as IMF dataflows.

<CodeGroup>

```python EconDB — single indicator
res = obb.economy.indicators(provider="econdb", symbol="PCOCO")
```

```python EconDB — CPI for multiple countries
res = obb.economy.indicators(
    symbol="CPI",
    country="united_states,jp",
    provider="econdb",
)
```

```python EconDB — main indicators bundle
res = obb.economy.indicators(
    provider="econdb",
    symbol="main",
    country="eu",
)
```

```python IMF — gold reserves by country
res = obb.economy.indicators(
    provider="imf",
    symbol="IL::RGV_REVS",
    country="*",
    frequency="month",
    limit=1,
    start_date="2025-09-30",
)
```

</CodeGroup>

<Note>
Use `obb.economy.available_indicators` to browse all supported indicator codes before querying.
</Note>

### `obb.economy.available_indicators`

List all economic indicators available from a given provider, along with metadata such as unit and frequency.

```python
res = obb.economy.available_indicators(provider="econdb")
```

### `obb.economy.country_profile`

Get a snapshot of country-level statistics and economic indicators in a single call.

<CodeGroup>

```python Single country
res = obb.economy.country_profile(provider="econdb", country="united_kingdom")
```

```python Multiple countries — full history
res = obb.economy.country_profile(
    country="united_states,jp",
    latest=False,
    provider="econdb",
)
```

</CodeGroup>

## GDP

### `obb.economy.gdp.nominal`

Nominal GDP by country. Filter by units (total or per capita) and frequency.

<CodeGroup>

```python Default
res = obb.economy.gdp.nominal(provider="oecd")
```

```python Per capita, all countries, annual
res = obb.economy.gdp.nominal(
    units="capita",
    country="all",
    frequency="annual",
    provider="oecd",
)
```

</CodeGroup>

### `obb.economy.gdp.real`

Real GDP data adjusted for inflation.

<CodeGroup>

```python OECD default
res = obb.economy.gdp.real(provider="oecd")
```

```python EconDB — multiple countries
res = obb.economy.gdp.real(
    country="united_states,germany,japan",
    provider="econdb",
)
```

</CodeGroup>

### `obb.economy.gdp.forecast`

Forward-looking GDP projections, including per-capita options and annual or quarterly frequency.

```python
res = obb.economy.gdp.forecast(
    country="united_states,germany,france",
    frequency="annual",
    units="capita",
    provider="oecd",
)
```

## Consumer prices

### `obb.economy.cpi`

Consumer Price Index (CPI) data by country. Apply transformations for year-over-year or period comparisons.

<CodeGroup>

```python FRED — multiple countries
res = obb.economy.cpi(country="japan,china,turkey", provider="fred")
```

```python OECD — period change
res = obb.economy.cpi(
    country="united_states,united_kingdom",
    transform="period",
    provider="oecd",
)
```

```python IMF — CPI basket weightings
res = obb.economy.cpi(
    provider="imf",
    country="CAN",
    transform="weight_percent",
    expenditure="all",
    limit=1,
)
```

</CodeGroup>

### `obb.economy.pce`

Personal Consumption Expenditures (PCE) reports from FRED, including the PCE price index.

```python
res = obb.economy.pce(
    provider="fred",
    date="2024-05-01,2024-04-01,2023-05-01",
    category="pce_price_index",
)
```

### `obb.economy.retail_prices`

Average retail prices for common consumer items from FRED, with optional regional and category filtering.

<CodeGroup>

```python Eggs — northeast region
res = obb.economy.retail_prices(
    item="eggs",
    region="northeast",
    provider="fred",
)
```

```python Meats — year-over-year change
res = obb.economy.retail_prices(
    item="meats",
    transform="pc1",
    provider="fred",
)
```

</CodeGroup>

## Interest rates and monetary policy

### `obb.economy.interest_rates`

Country-level interest rates from OECD: overnight (immediate), short (3-month), or long (10-year).

<CodeGroup>

```python Short-term rates — all countries, quarterly
res = obb.economy.interest_rates(
    provider="oecd",
    country="all",
    duration="immediate",
    frequency="quarter",
)
```

```python Long-term rates — UK and Germany
res = obb.economy.interest_rates(
    duration="long",
    country="united_kingdom,germany",
    frequency="monthly",
    provider="oecd",
)
```

</CodeGroup>

### `obb.economy.money_measures`

M1 and M2 money supply measures and their components from the Federal Reserve's H.6 release.

```python
res = obb.economy.money_measures(provider="federal_reserve")
res_unadjusted = obb.economy.money_measures(adjusted=False, provider="federal_reserve")
```

### `obb.economy.central_bank_holdings`

Balance sheet holdings of a central bank, including Treasuries and agency securities.

<CodeGroup>

```python Latest Treasury holdings
res = obb.economy.central_bank_holdings(provider="federal_reserve")
```

```python Historical summary
res = obb.economy.central_bank_holdings(
    provider="federal_reserve",
    summary=True,
)
```

```python Agency debt as of a historical date
res = obb.economy.central_bank_holdings(
    provider="federal_reserve",
    holding_type="agency_debts",
)
```

</CodeGroup>

### `obb.economy.fomc_documents`

FOMC statements, minutes, and other Federal Reserve policy documents by year and type.

```python
res = obb.economy.fomc_documents(
    provider="federal_reserve",
    year=2022,
    document_type="minutes",
)
```


Retrieve time series data for one or more BLS series IDs.


## FRED utilities

### `obb.economy.fred_series`

Retrieve one or more FRED time series by symbol, with optional transformation.

```python
res = obb.economy.fred_series(
    symbol="CBBTCUSD",
    transform="pc1",
    provider="fred",
)
```

### `obb.economy.fred_search`

Search for FRED series or economic releases by ID or keyword. Returns metadata only — use series IDs with `fred_series`.

```python
res = obb.economy.fred_search(provider="fred")
```

### `obb.economy.fred_regional`

Query the Geo Fred API for regional economic data by series group ID. Use `fred_search` with `series_id` to find group IDs.

```python
res = obb.economy.fred_regional(
    symbol="NYICLAIMS",
    start_date="2021-01-01",
    end_date="2021-12-31",
    limit=10,
    provider="fred",
)
```

### `obb.economy.fred_release_table`

Get economic release data by release ID and optional element ID from FRED.

```python
res = obb.economy.fred_release_table(
    release_id="50",
    element_id="4881",
    provider="fred",
)
```
