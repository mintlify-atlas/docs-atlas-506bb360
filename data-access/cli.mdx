The OpenBB CLI is an interactive terminal interface that wraps the OpenBB Platform. Navigate menus, fetch data, and run routine scripts — all from your terminal.

## Installation

```bash
pip install openbb-cli
```

## Launching the CLI

<CodeGroup>

```bash openbb command
openbb
```

```bash Python module
python -m openbb_cli
```

</CodeGroup>

Both commands print `Loading...` and then open the interactive prompt.

### Launch flags

| Flag | Description |
|---|---|
| `--dev` | Launch in developer mode with additional logging. |
| `--debug` | Enable debug-level output for troubleshooting. |

```bash
openbb --dev
openbb --debug
```

## Navigating menus

The CLI uses a path-like navigation model. Type a menu path to move into it, or call a command directly from the root:

```
(openbb) /equity/price/historical --symbol AAPL --provider fmp
```

You can also navigate step by step:

```
(openbb) /equity
(equity) /price
(equity/price) historical --symbol AAPL --provider fmp
```

Use `..` to go up one level:

```
(equity/price) ..
(equity) 
```

Use `/` to return to the root:

```
(equity/price) /
(openbb)
```

## Tab completion

Press `Tab` to complete commands, menu paths, and parameter names. Double-tap `Tab` to list all available options at the current level.

```
(openbb) /equity/price/  <Tab>
historical   nbbo   performance   quote
```

## Configuration

The CLI reads settings from `~/.openbb_platform/`. Key files:

| File | Purpose |
|---|---|
| `user_settings.json` | API keys, default provider, and preferences. |
| `system_settings.json` | System-level configuration. |

Example `user_settings.json`:

```json
{
  "credentials": {
    "fmp_api_key": "YOUR_FMP_KEY",
    "intrinio_api_key": "YOUR_INTRINIO_KEY"
  },
  "preferences": {
    "data_directory": "~/OpenBBUserData",
    "export_directory": "~/OpenBBUserData/exports"
  }
}
```

<Tip>
Run `/settings` inside the CLI to view and update preferences interactively.
</Tip>

## Exporting data

Append `--export` to any command to save results to a file:

```
(openbb) /equity/price/historical --symbol AAPL --provider fmp --export csv
```

Exported files are saved to the configured `export_directory` (default: `~/OpenBBUserData/exports`).
